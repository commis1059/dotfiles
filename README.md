# My Dotfiles Based on MItamae
[MItamae](https://github.com/itamae-kitchen/mitamae) is a configuration management tool.

## System requirements
* macOS

## Installation
```
$ git clone https://github.com/commis1059/dotfiles.git
$ cd dotfiles
$ ./init.sh
```
